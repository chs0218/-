---구현 내용--
' Enter ': 육면체 생성
' o ': 직각투영으로 화면을 띄웁니다. (메인 화면만 변화)
' p ': 원근투영으로 화면을 띄웁니다. (메인 화면만 변화)
' M/m ': 육면체가 멈춰 있을경우 움직이고 움직이고 있을 경우 멈춥니다.
	(각 육면체의 속도와 높이의 최대, 최소값은 랜덤하게 설정했습니다.)
' x ': 카메라가 바닥의 x축을 기준으로 양의 방향 회전 / 중지
' X ': 카메라가 바닥의 x축을 기준으로 음의 방향 회전 / 중지
' y ': 카메라가 바닥의 y축을 기준으로 양의 방향 회전 / 중지
' Y ': 카메라가 바닥의 y축을 기준으로 음의 방향 회전 / 중지
' R/r ': 미로를 생성
' E/e ': 객체 자동이동 실행 / 중지
' V/v ': 육면체가 움직이고 있을 경우 중지 시키면서 낮은 높이로 변환, 육면체가 정지해 있을 경우 움직이게 만듦
' w/a/s/d, W/A/S/D ': 1인칭의 경우 객체 기준으로 이동, 3인칭의 경우 월드 기준으로 이동
' +/- ': 객체 속도 증가/감소
' , ' : 3인칭 시점(쿼터뷰)
' . ' : 1인칭 시점(객체시점)
' c ': 초기화
' q ': 프로그램 종료
' Tab ': 입력위치 변화
' 1/2/3/4/5/6/7/8/9/0 ': 숫자 입력
' BackSpace ': 입력 수정

--- 보너스 구현 요소 ---
프로그램 좌측 상단에서 입력을 받고 입력 받은 값을 확인하고 수정할 수 있습니다. 
Caret을 통해 현재 입력위치를 보여줍니다.
클릭 or Tab을 이용해 입력 위치를 바꿀 수 있습니다. (클릭의 경우 윈도우 크기를 변화시키면 작동이 이상할 수 있습니다.)
우측 상단의 미니맵의 경우 'V/v' 명령으로 낮은 높이가 아니면 객체가 잘 보이지 않아 우측 하단에 객체를 중심으로 한 미니맵을 출력했습니다.
객체를 'w/a/s/d, W/A/S/D'로 이동시킬 때 3인칭 쿼터뷰에서는 월드좌표를 기준으로 w/a/s/d로 움직이지만 1인칭 객체 시점에서는 객체를 기준으로 이동시키도록 구현하였습니다.

--- 기타 알아야 할 요소 ---
미로는 5 ~ 40으로 생성 가능하게 만들었습니다. 숫자가 커야 미로를 만들었을 때 볼 만한 미로가 만들어집니다.
콘솔창에 키보드 명령어를 출력했습니다.
